Final Project

This is my final project for CSC 106. I wanted to make a page dedicated to something that I enjoy.

I decided to make a page dedicated to the 4-team College Football Playoff Era, which is coming to a close this year, and is being replaced by a 12-team playoff. Feel free to check it out and learn about this era of college football.

In order to navigate the project, use the buttons provided in order to change between pages. Each page has its own information about the playoff. The user is able to navigate between these pages, and discover new information about the history of the 4-team playoff
